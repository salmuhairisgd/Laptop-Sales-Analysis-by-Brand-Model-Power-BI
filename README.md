## Project Overview
An interactive Power BI Dashboard designed to analyze laptop sales data across various brands and technical specifications. This project provides a comprehensive look at market trends, consumer preferences for storage types (SSD vs. HDD), and performance ratings for top-tier tech brands like MSI, Apple, Dell, and HP.

## Key Insights
**Product Performance:** Detailed sales analysis for over 973 laptop units, identifying the top 5 models by total sales value.

**Brand Ratings:** Comparative analysis of the top 5 brands based on average user ratings, with MSI and ASUS leading the list.

**Hardware Trends:** Visualization of storage preferences, highlighting the dominance of different capacities (from 32GB to 2TB) and storage technologies.

**Specification Filtering:** Ability to segment sales based on "Touch Screen" features and storage configurations.

## Tech Stack
Power BI: Data visualization and multi-layered dashboard design.

Power Query: Managing and cleaning complex product specification datasets.

Data Modeling: Creating relationships between brands, models, and technical specs.

## How to Use
To explore the interactive report:

1- Download the `Laptop sales brand` file from this repository.

2- Open the file using **Power BI Desktop**.

3- Filter the data using the Touch Screen or Storage Type slicers at the top to see how specifications affect sales and ratings.

4- Explore the Top 5 Models chart to see which high-end laptops (like Dell Alienware or Apple MacBook Pro) are driving revenue.
